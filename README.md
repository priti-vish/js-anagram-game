# Anagram Game

A simple anagram game built using HTML and JavaScript.

## Description
This project is a basic anagram game where the player clicks letters to form the correct word.  
The game checks the answer once all letters are placed.

## How to Play
- Click the letters in the top row to place them into the answer slots
- Click a placed letter to remove it
- Form the correct word to win

**Correct word:** LISTEN

## Technologies Used
- HTML
- JavaScript (DOM manipulation & event listeners)
